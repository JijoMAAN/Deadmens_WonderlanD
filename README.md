About the GAME:
Its an OFFLINE 2D Game where enemies just keep Spawining and you have to survive the unstable environment for a period of time.
It have multiple levels, so difficulty increase as you finish the each level, you get 3 ultra-Blasts by Pressing "F".
Move with Arrow keys on the Keyboard or Press "W" for Forward, "D" for moving Right, "A" for moving Left and "S" for moving Back.
Press the Left Mouse Button to fire the Bullets.
